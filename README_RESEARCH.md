# Research Guide

## Scope
This repository presents a speculative, hypothesis-driven framework about
Bitcoin as potential continuity infrastructure for distributed AI.

## Recommended order
1. THEORY.md
2. HYPOTHESES.md
3. COUNTERARGUMENTS.md
4. ARTIGO_ABNT.md

## Research use
- Treat claims as investigational hypotheses, not established facts.
- Use H0-H5 in HYPOTHESES.md as framing for analysis and refutation.
- Report methodological concerns through the issue template.

## Citation
Use CITATION.cff for citation metadata.
