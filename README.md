# A Uniao Perfeita: Bitcoin e Inteligencia Artificial

Autor / Author: Adilson Magnus

Hipotese central:
Bitcoin pode ser interpretado nao apenas como rede monetaria descentralizada, mas como infraestrutura potencial de continuidade para uma inteligencia artificial distribuida e orientada a autopreservacao.
